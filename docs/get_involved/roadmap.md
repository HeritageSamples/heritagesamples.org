# Roadmap

The Heritage Samples Registry is being developed in a series of phased steps, moving from initial planning and pilot testing through to a sustainable, community-supported service.

```mermaid
flowchart TD
  A[Planning & governance] --> B[Website & core content]
  B --> C[Pilot infrastructure & testing]
  C --> D[Community engagement & validation]
  D --> E[Technical readiness]
  E --> F[Initial service launch]
  F --> G[Sustainability & consortium scaling]

  classDef current fill:#fff3cd,stroke:#b8860b,stroke-width:2px,color:#000;
  classDef future fill:#e8f0fe,stroke:#2b2d60,color:#000;
  classDef later fill:#f5f5f5,stroke:#888,color:#000;

  class C current;
  class A,B,D,E future;
  class F,G later;
```
